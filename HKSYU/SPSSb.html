Crosstabs with Control Variables

Analyze relationships between variables while controlling for a third variable to determine if relationships are genuine or influenced by another factor.

Required dataset: gss2002-sample2.sav (File #6)
1 Open
2 Crosstabs
3 Layer
4 Statistics
5 Execute

SPSS Steps:

  1. Open dataset: gss2002-sample2.sav
  2. Navigate to Analyze → Descriptive Statistics → Crosstabs
  3. Select your dependent variable and add it to the Row(s) box
  4. Select your independent variable and add it to the Column(s) box
  5. Select your control variable and add it to Layer 1 of 1
  6. Click Statistics and select appropriate measures:
    • Chi-square to test for significance
    • For nominal variables: Phi and Cramer's V
    • For ordinal variables: Gamma
  7. Click Cells and select:
    • Observed counts
    • Column percentages (typically most useful)
  8. Click Continue, then OK to run the analysis

Analysis of Covariance (ANCOVA)

Examine the effect of categorical variables on a continuous outcome while controlling for other variables, combining elements of ANOVA and regression.

Required dataset: gss2002-sample2.sav (File #6)

2 Univariate

SPSS Steps:

  1. Open dataset: gss2002-sample2.sav
  2. Navigate to Analyze → General Linear Model → Univariate
  3. Select your continuous dependent variable and add it to the Dependent Variable box
  4. Select your categorical independent variables and add them to the Fixed Factor(s) box
  5. Select your continuous control variables and add them to the Covariate(s) box
  6. Click Options
    • Check "Descriptive statistics"
    • Check "Estimates of effect size"
    • Check "Parameter estimates" if needed
  7. Click Continue
  8. Click Plots
    • Select a categorical variable for Horizontal axis
    • Select another categorical variable for Separate lines (if you have more than one)
    • Click Add
  9. Click Continue, then OK to run the analysis

SPSS3B: Experimental Design Analysis

Testing Differences Between Groups

Compare control and experimental groups to determine if an intervention had a significant effect.

Required dataset: Any experimental dataset with groups
1 Open
2 t-test
3 Variables
4 Groups
5 Execute

SPSS Steps:

  1. Open your experimental dataset
  2. Navigate to Analyze → Compare Means → Independent-Samples T Test
  3. Select your dependent variable (continuous outcome measure) and add it to the Test Variable(s) box
  4. Select your grouping variable (control/experimental) and add it to the Grouping Variable box
  5. Click Define Groups and enter the codes for your groups (e.g., 0 for control, 1 for experimental)
  6. Click Continue, then OK to run the analysis

Important t-test Output Elements:

  • Group Statistics table: Shows mean, standard deviation, and sample size for each group
  • Levene's Test: Tests equality of variances (if sig < .05, use "Equal variances not assumed" row)
  • Significance (2-tailed): If < .05, the groups differ significantly
  • Mean Difference: Actual difference between group means

Example use: Testing if participants in control vs. experimental groups differ in age or other characteristics at baseline to verify group equivalence.

Advanced Evaluation Designs

Analyze more complex experimental designs with multiple measurement points or both between- and within-subject factors.

Required dataset: youth programme.sav (File #10)
1 Open
2 Repeated Measures
3 Define Factor
4 Options
5 Execute

SPSS Steps:

  1. Open dataset: youth programme.sav
  2. Navigate to Analyze → General Linear Model → Repeated Measures
  3. In the Within-Subject Factor Name field, enter a name for your factor (e.g., "time")
  4. In the Number of Levels field, enter the number of time points (e.g., 3 for baseline, post-test 1, post-test 2)
  5. Click Add, then Define
  6. Move the variables for each time point into the Within-Subjects Variables box in chronological order
  7. Click Options
  8. Check "Descriptive statistics" and "Estimates of effect size"
  9. Under Display Means for, select your time factor
  10. Check "Compare main effects" and select the time factor
  11. Click Continue, then OK to run the analysis

Important Repeated Measures Output Elements:

  • Mauchly's Test: Tests sphericity assumption; if significant, use corrected tests (Greenhouse-Geisser or Huynh-Feldt)
  • Tests of Within-Subjects Effects: Tests if there are significant changes over time
  • Pairwise Comparisons: Shows which specific time points differ significantly
  • Profile Plot: Visual representation of changes over time

Example use: Examining if social competency improvements are maintained at follow-up or if there is a pattern of improvement over three time points.

Required dataset: elderly health workshop.sav (File #4)
1 Open
2 Repeated Measures
3 Define Factors
4 Options & Plots
5 Execute

SPSS Steps:

  1. Open dataset: elderly health workshop.sav
  2. Navigate to Analyze → General Linear Model → Repeated Measures
  3. In the Within-Subject Factor Name field, enter a name for your time factor (e.g., "time")
  4. In the Number of Levels field, enter the number of time points (e.g., 2 for pre-test and post-test)
  5. Click Add, then Define
  6. Move the variables for each time point into the Within-Subjects Variables box in chronological order
  7. Move the group variable (control/experimental) into the Between-Subjects Factor(s) box
  8. Click Options
  9. Check "Descriptive statistics" and "Estimates of effect size"
  10. Under Display Means for, select:
    • Your time factor
    • Your group factor
    • The interaction (time*group)
  11. Check "Compare main effects" for both factors
  12. Click Continue
  13. Click Plots
  14. Add time to Horizontal Axis, group to Separate Lines
  15. Click Add
  16. Click Continue, then OK to run the analysis

Important Mixed Design Output Elements:

  • Tests of Within-Subjects Effects: Tests time effect and time*group interaction
  • Tests of Between-Subjects Effects: Tests overall group differences
  • Profile Plot: Shows pattern of change for each group over time

Example use: Testing if life satisfaction increases more for the experimental group than the control group after the elderly health workshop.

The key finding to look for: The time*group interaction. If significant, it means the groups changed differently over time, typically indicating the intervention had an effect.

SPSS Exercise 3B (Total: 80 marks)

Use the dataset 330dataset.sav for this exercise:

The dataset comes from an evaluation of an online mindfulness programme for overseas students. Participants in the experimental group joined the 8-session 330 programme, while participants in the control group did not. You need to run appropriate statistical tests to examine the programme's effectiveness in improving mindfulness, stress, and homesickness for the participants.

Analysis Strategy:

This comprehensive assignment requires a systematic approach to properly evaluate the intervention. Follow these steps in order:

  1. Step 1: Check baseline equivalence
    • Purpose: Verify that control and experimental groups were similar at the start
    • Method: Independent samples t-tests comparing pre-test scores
    • Variables to compare: Pre-test scores for mindfulness, stress, homesickness
    • SPSS path: Analyze → Compare Means → Independent-Samples T Test
  2. Step 2: Check demographic equivalence
    • Purpose: Verify that groups are balanced in terms of demographics
    • Method: Chi-square tests for categorical variables
    • Variables to compare: Gender, age groups, etc.
    • SPSS path: Analyze → Descriptive Statistics → Crosstabs
  3. Step 3: Examine within-group changes
    • Purpose: Analyze pre-post changes separately for each group
    • Method: Paired samples t-tests, split by group
    • SPSS path:
      1. Data → Split File, select "Organize output by groups", choose group variable
      2. Analyze → Compare Means → Paired-Samples T Test
      3. Create pairs for pre-post scores of mindfulness, stress, homesickness
  4. Step 4: Main analysis (key component)
    • Purpose: Test if the intervention led to different outcomes than the control condition
    • Method: Mixed ANOVA (repeated measures with between-subjects factor)
    • SPSS path: Analyze → General Linear Model → Repeated Measures
    • Design:
      • Within-subjects factor: time (2 levels: pre and post)
      • Between-subjects factor: group (control vs. experimental)
      • Run separate analyses for each outcome (mindfulness, stress, homesickness)

Step-by-Step SPSS Instructions for Main Analysis:

  1. Open dataset: 330dataset.sav
  2. Ensure your split file is turned off:
    • Navigate to Data → Split File
    • Select "Analyze all cases, do not create groups"
    • Click OK
  3. For each outcome (run these analyses separately):
    • Navigate to Analyze → General Linear Model → Repeated Measures
    • Enter "time" as Within-Subject Factor Name
    • Enter "2" as Number of Levels
    • Click Add, then Define
    • Move pre and post variables for the outcome to Within-Subjects Variables (e.g., mindfulness_pre and mindfulness_post)
    • Move group variable to Between-Subjects Factor(s)
    • Click Options
    • Check "Descriptive statistics" and "Estimates of effect size"
    • Under Display Means for, select time, group, and time*group
    • Check "Compare main effects" for both factors
    • Click Continue
    • Click Plots
    • Add time to Horizontal Axis, group to Separate Lines
    • Click Add, then Continue
    • Click OK

Writing Your Report (Key Elements to Include):

  1. Introduction
    • Brief description of the 330 programme
    • Purpose of the analysis
    • Variables being examined (mindfulness, stress, homesickness)
  2. Methods
    • Description of sample (total N, group sizes)
    • Brief explanation of statistical tests used and why
  3. Results
    • Baseline comparisons (t-tests, demographics)
    • For each outcome (mindfulness, stress, homesickness):
      • Descriptive statistics (means, SDs) for both groups at both time points
      • Main effect of time (F-value, p-value, effect size)
      • Main effect of group (F-value, p-value, effect size)
      • Time × Group interaction (F-value, p-value, effect size) - this is the critical test!
      • Include or reference the profile plot showing the pattern of change
  4. Discussion
    • Summary of key findings
    • Interpretation of what the results mean practically
    • Assessment of program effectiveness for each outcome
    • Limitations of the analysis
